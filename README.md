# Landing B2B — Reboot Academy

Landing de ventas directa para agencias de marketing. Stack: HTML/CSS/JS vanilla, deploy en Vercel.

## Como adaptar para una campana nueva

Abre Claude Code en este proyecto y escribe:

```
/nueva-campana
```

Claude te preguntara 5 datos antes de generar.

Tambien puedes escribir directamente:
"Quiero adaptar la landing para [ciudad/campana]"

## Deploy

Push a main → Vercel despliega automatico.
